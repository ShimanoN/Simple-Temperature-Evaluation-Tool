# CHANGELOG

このプロジェクトの主要な改修履歴を時系列でまとめます。

## 2026-02-27 — リファクタリング & バグ修正 (Rev.2)
- 単一チャネル（1センサ）設計へ正式に移行。
- MAX31855（熱電対）読み取りで一時的に発生していた異常値を防ぐため、
  - `readCelsius()` の異常値検出時に最大3回のリトライを実装。
  - SD書き込み後に SPI バスをリセットする処理を追加（`SPI.end(); SPI.begin();`）。
- SD カード初期化時の CS ピン指定を明確化（microSD CS = GPIO4）。
- Welford 法による統計（平均・分散）を `Logic_Task` 内で定期的に更新し、
  RUN 中の CSV 出力に正しい平均・標準偏差が含まれるよう修正。
- SD 書き込みタイミングを調整（初回書き込み条件を `D_Count >= 10` に変更）。
- 各種ドキュメントを更新・整理（`README.md`, `specs/*`, `docs` 内の目次等）。

## 2026-02-26 — 前段のリファクタリング (Rev.1)
- グローバル定数・命名規則の整理。
- UI 描画ロジックの部分更新化（行ベース描画）を導入。
- EEPROM 設定の分離（`EEPROMManager` の独立化）。
- ドキュメントの統廃合（旧リファクタリング計画のアーカイブ化）。
