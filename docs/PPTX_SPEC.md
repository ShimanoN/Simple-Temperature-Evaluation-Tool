## 共通レイアウトルール

### 座標系
- 単位: インチ (Inches)
- スライド幅: 13.33"、高さ: 7.5"
- **ヘッダーバー**: 左上 (x=0, y=0)、幅=13.33"、高さ=1.0"（塗り=ネイビー）
  - スライドタイトル: ヘッダー内 y=0.08" から、フォント24pt、太字、白
  - セクションラベル: ヘッダー内 y=0.65" から、フォント12pt、オレンジ
- **フッターバー**: 左上 (x=0, y=6.9")、幅=13.33"、高さ=0.6"（塗り=ネイビー）
  - ページ番号: フッター内右端、フォント11pt、白
  - プロジェクト名: フッター内左端、フォント11pt、オレンジ
- **コンテンツエリア**: y=1.1" 〜 y=6.85"（高さ=5.75"）、x=0.35" 〜 x=12.98"（幅=12.63"）

### 色定数
```
C_NAVY      = RGBColor(0x0D, 0x47, 0x7F)  # ヘッダー・フッター
C_ORANGE    = RGBColor(0xFF, 0x6D, 0x00)  # アクセント

```

### テーブルスタイル
- **ヘッダー行**: 塗り=C_NAVY、文字=白、太字、フォント13pt
- **データ行(偶数)**: 塗り=C_GRAY_LT、文字=C_DARK_TXT、フォント12pt
- **データ行(奇数)**: 塗り=白、文字=C_DARK_TXT、フォント12pt
- **最大行数**: 1テーブルあたり10行まで（超える場合は分割）
- **行高**: 0.45"（最小）〜 0.5"（標準）

### フォントサイズルール
- スライドタイトル: 24pt 太字
- セクションラベル: 12pt
- 見出し（コンテンツ内）: 18pt 太字
- 本文・箇条書き: 14-15pt
- テーブルヘッダー: 13pt 太字
- テーブルデータ: 12pt
- 注釈・補足: 12pt 斜体 or 通常
- **絶対禁止**: 10pt 未満（読み取り不可）

### フォントファミリー

| 用途 | フォント名 | 選定理由 |
|------|-----------|----------|
| 見出し・英数字タイトル | `Segoe UI` Bold | Windows 標準の幾何学系サンセリフ。ネイビー背景との相性◎ |
| 日本語テキスト全般 | `游ゴシック` | Windows 10+ 標準。Meiryo より字面が小さく洗練。英数字にも対応 |
| コード・定数・ファイル名 | `Consolas` | 等幅。VS Code と同じフォントで技術資料として自然 |
| KPI 数値・大きな数字 | `Segoe UI Semibold` | 力強さと視認性を両立。カード・サマリーの数値に使用 |

> **実装メモ**: 通常テキストは `游ゴシック` を基本フォントに設定（日英混在でも破綻しない）。ヘッダーバー・タイトルの英字など視覚的に英語が主体の箇所は `Segoe UI` を個別 run で指定する。

### レイアウト禁止事項
- テキストボックス・テーブルが y=6.85" を超えてはならない
- テキストボックスが互いに重複してはならない（座標計算を徹底する）
- 1スライドにテーブルは最大1個（例外：比較スライドのみ2個）
- 1スライドの箇条書きは最大6項目

## Python スクリプト設計メモ

### 守るべきルール
1. `add_textbox()` を呼ぶ前に **座標・サイズを厳密に計算**して重複を禁止
2. テーブルは `make_table()` 関数で生成し、**必ず列幅の合計 ≤ コンテンツ幅**を確認
3. 1 スライドの生成後に **続く y 座標が 6.85" を超えていないか** 確認
4. フォントは `"游ゴシック"` を基本（日英混在対応）、英字見出しは `"Segoe UI"` 、コード表示は `"Consolas"` を使用
5. 各スライドを生成する関数は **引数 `prs` のみ**（状態をグローバルに持たない）
6. スクリプト実行後、スライド枚数が **23 枚**であることをアサート

### ファイル構成
```
docs/career/
  create_presentation_v2.py   ← 新規作成（旧 create_presentation.py は残す）
  Simple_Temp_Tool_Overview_v2.pptx  ← 出力先
```

### 主要ヘルパー関数
- `blank_slide(prs)` → 白背景のブランクスライドを追加
- `header_bar(slide, title, section_label)` → ネイビーヘッダーバー + テキスト
- `footer_bar(slide, page_num)` → ネイビーフッターバー + ページ番号
- `add_textbox(slide, text, x, y, w, h, ...)` → テキストボックス
- `make_table(slide, x, y, total_w, rows, col_widths, ...)` → テーブル生成
- `add_rect(slide, x, y, w, h, fill_color, line_color, line_width)` → 矩形
- `lcd_mock(slide, x, y, w, h, lines, border_color)` → LCD 画面風ブロック
